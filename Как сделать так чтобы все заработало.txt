Сделал сервис API побыстрому. не стал брать то что у вас выложено потому сто хотел сделать что то на NET без CORE - получилось в первом приближении :)))

как сделать так чтобы получилось:

- 1 - 
Собираем API:
a) открыть проект Service\JobM_NET.sln в Visual Studio (я использовал бесплатную VS 2019)
b) в фапйле web.config поправить строку для подключения к BBDD на актуальную
с) Собрать. Если возникли ошибки то связаться со мной, но я собрал все без прооблем.
d) Запустить проект. После запуска откроется ваш навигатор по умочанию и отобразится стартовая страничка сервиса. Нам необходимо запомнить url по которому она открылась. на моем компьютере это https://localhost:44311
ОТЛАДКУ НЕ ПРЕРЫВАЕМ - будем иcпользовать API после сборки клиента


- 2 - 
Собираем клиента.
a)	открываем в VS (или чего у вас там незнаю) папку Client и выполняем команду в терминале
	npm install
	сидим и ждем когда все установится
	

b) настраиваем где у нас API - открываем файл Clientenments\environment.ts

export const environment = {
  production: false,
  apiUrl: 'https://localhost:44311', // Вот сюда вписываем url который запоминали ранее          
  unknown_user: 'assets/img/unknown_user.png',
  signin_user: 'assets/img/signin_user.png'
};
и обязательно сохраняем все измененные файлы

с) Собираем приложение с помощью команды 
	ng serve
	сидим ждем пока соберется...

После удачной сборки будет отображен url откуда надо запускать клиента  - запускаем!!!!


ВНИМАНИЕ по умолчанию BBDD пустая (пользователей нет) и при первом же логине создаем пользователя которым логинимся потом.
